# SmartDoc-AI (Phase 1)

## Description
A local Large Language Model (LLM) chat application built using Hugging Face Transformers.
The project demonstrates model loading, text generation, and interactive CLI-based chat.

## Tech Stack
- Python
- Hugging Face Transformers
- PyTorch

## Features
- Local LLM inference (GPT-2)
- Command-line chat interface
- Model download & caching
- Token-controlled generation

## Status
Phase 1 completed.  
Future work: LangChain integration, memory, RAG, API deployment.

pip install -r requirements.txt
